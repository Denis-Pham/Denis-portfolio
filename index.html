About

Numbers tell stories. I help people listen.

A short intro — who I am, what I care about, and why someone should hire me.

DP

I run the performance system for Hasaki Vietnam's Cosmetics division, end to end: I write the SOPs that define how stores should operate, turn them into daily task workflows assigned to store teams, then measure the results with the KPI framework I designed — the same numbers that drive monthly performance reviews. Because one person owns the whole loop, the standards, the tasks, and the scores always agree with each other.

That system thinking comes from a decade in quality: six years of QA/QC across manufacturing, retail and education — internal audits, root-cause analysis, quality documentation — then project management at a game studio before moving into retail performance. My toolbox is Excel (deeply), Python pandas, and increasingly Claude Code for shipping internal tools in a single sitting.

Outside of work I'm learning web development one project at a time — data dashboards, a landing page for an AI music channel, document automation tools. The common thread: take messy reality, turn it into something people can act on.
